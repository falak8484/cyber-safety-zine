# 📖 Cyber Safety Zine — Digital Awareness Magazine

Cyber Safety Zine is a visually engaging digital magazine designed to spread awareness about online safety, cybersecurity practices, and everyday digital risks.

It transforms important security concepts into simple, readable, and aesthetic content for a wider audience.

---

## 🌐 Live Zine

👉 https://falak8484.github.io/cyber-safety-zine/

---

## 🧠 Purpose

Cybersecurity is not just for experts — it affects everyone.

This project aims to:
- simplify complex security topics  
- promote safe digital habits  
- educate users in a visually appealing way  
- make cybersecurity accessible to beginners  

---

## 📚 Topics Covered

- 🔐 Password safety and best practices  
- 🎣 Phishing awareness  
- 🌐 Safe browsing habits  
- 📱 Social media privacy  
- 💻 Device security basics  
- 🚨 Common cyber threats  

---

## 🎨 Features

- 📖 Magazine-style layout  
- 🌸 Aesthetic and clean UI design  
- 📱 Responsive across devices  
- 🧠 Easy-to-understand explanations  
- 🎯 Focus on awareness and usability
